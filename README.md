# 🚀 7 Days Challenge – Build > Learn > Iterate

## 🎯 Objective
This is a 7-day intensive challenge focused on:
- Building real-world projects using **Web Development + ML + GenAI + Agentic AI**
- Strengthening **problem-solving and debugging skills**
- Learning and applying **Advanced GitHub workflows**
- Developing a **builder mindset** by shipping daily

> ⚡ Focus: Execution over perfection

---

## 🛠️ Core Focus Areas

### 💻 Development
- Backend (Node.js, APIs)
- Frontend (HTML/CSS/JS or React)
- Database (SQL)

### 🤖 AI / ML
- Generative AI APIs
- Prompt engineering
- Intro to Agentic AI systems

### ⚙️ Tools & Workflow
- Git & GitHub (advanced usage)
- GitHub Actions (CI/CD basics)
- SSH setup and usage

---

## 🔄 Daily Workflow (Flexible)

Each day follows a **build-driven loop**:

1. **Learn**
   - Pick concepts while building (Node.js, APIs, SQL, etc.)
   - Learn only what is needed for the current task

2. **Build**
   - Implement features
   - Focus on progress, not perfection

3. **Debug**
   - Solve errors and blockers
   - Understand *why* things break

4. **Explore**
   - Study at least one real-world project
   - Observe how features are implemented

5. **Reflect**
   - Track progress and insights
